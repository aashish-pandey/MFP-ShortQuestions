Q1. What is microprocessor?

Ans:
Microprocessor is a programmable, clock driven, register based electronic device that reads binary information from storage device called memory, accepts binary data as input and produces binary result as output.

Q2. What is difference between CPU and microprocessor?

Ans:
CPU is capable of performing all kinds of computing and arithematic functions while microprocessor deals in BIOS and memory circuit in addition to performing all CPU functions.

Q3. Define bit, byte, word and instruction.

Ans:
Bit: A binary digit(0 and 1) is called a bit.
Byte: Group of 8 bits is called a byte.
Word: Group of 16 bits is called a word.
Instruction: Instruction is a command given to computer to perform a specified operation on given data.

Q4. Discuss the function of ALU of 8085.

Ans:
ALU of 8085 performs arithematic and logic operations.

Q5. List four categories of 8085 instructions that can manipulate data.

Ans:

  1. Data Transfer Group: E.g. MOV, MVI, LXI
  2. Arithmetic Group: E.g. ADD, SUB, INR
  3. Logical Group: E.g. ANA, XRA, CMP
  4. Branch Group: E.g. JMP, JNZ, CALL

Q6. What are different addressing modes in 8085.

Ans:

  1. Immediate addressing mode
  2. Register addressing mode
  3. Direct addressing mode
  4. Indirect addressing mode
  5. Implied addressing mode

Q7. Why is data bus bidirectional?

Ans:
Data bus is bidirectional because data flows in both directions, from microprocessor to memory or input/output device and from memory or input/output device to microprocessor.

Q8. Specify four control signals used by 8085 MPU.

Ans:

  1. RD
  2. WR
  3. ALE
  4. IO/M

Q9. What is the function of accumulator?

Ans:
Accumulator is used to store 8 bit data and is used to perform arithmetic and logic operations.

Q11. Discuss machine cycles and states.

Ans: The time required by microprocessor to complete an operation of accessing memory or input/output devices is called machine cycle.
One time period of frequency of microprocessor is called state.

Q12. What are various status flag provided in 8085?

Ans:

  1. Carry flag (Cy)
  2. Auxillary flag (AC)
  3. Sign flag (S)
  4. Parity flag(P)
  5. Zero flag(Z)
